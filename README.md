# degerkaybimi.com - Araç Değer Kaybı Hesaplama Sistemi

Bu proje, trafik kazası sonrası araçlarda oluşan değer kaybını hesaplamak ve kullanıcıları bilgilendirmek amacıyla oluşturulmuş bir web aracıdır.

## Özellikler
- **Hesaplama Motoru:** KM, rayiç bedel ve hasar durumuna göre tahmini tazminat hesabı.
- **Modern Tasarım:** Tailwind CSS kullanılarak mobil uyumlu (responsive) hale getirilmiştir.
- **WhatsApp Entegrasyonu:** Müşterilerin doğrudan uzmanlara ulaşmasını sağlar.

## Teknik Yapı
- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Hosting:** Vercel
- **Repository:** GitHub
